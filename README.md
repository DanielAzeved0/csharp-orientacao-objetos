# C#: Aplicando Orientação a Objetos

Um projeto de aprendizado em C# focado em aplicar conceitos de orientação a objetos, desenvolvido como parte do curso de formação em desenvolvimento .NET.

## 📋 Descrição do Projeto

Este projeto representa o aprofundamento em conceitos de orientação a objetos em C#. Ele foi criado com o objetivo de consolidar práticas fundamentais, como:

• Abstração e encapsulamento
• Herança e polimorfismo
• Composição de objetos
• Trabalho com coleções
• Estruturação de aplicações orientadas a objetos

## 🛠️ Tecnologias Utilizadas

• Linguagem: C#
• Framework: .NET 10.0
• IDE: Visual Studio

## 📁 Estrutura do Projeto

```
ScreenSound/
├── Program.cs           # Arquivo principal da aplicação
├── Banda.cs             # Classe para representar uma banda
├── Album.cs             # Classe para representar um álbum
├── Musica.cs            # Classe para representar uma música
├── Podcast.cs           # Classe para representar um podcast
├── Episodio.cs          # Classe para representar um episódio
├── ScreenSound.csproj   # Configuração do projeto
└── bin/                 # Arquivos compilados (gerados)
```

## 🚀 Como Executar

1. Clone ou baixe o repositório
2. Navegue até a pasta do projeto: `cd ScreenSound`
3. Compile e execute: `dotnet run`

## 📝 Requisitos

• .NET 10.0 ou superior instalado
• Visual Studio ou VS Code (opcional)
